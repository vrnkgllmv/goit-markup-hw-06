# WebStudio

This project is a simple HTML/CSS layout for the "WebStudio" website, created as part of a homework assignment.

## Structure

- `index.html`: Main HTML file containing the website markup.
- `images/`: Folder containing team member images and other assets.

## Features

- Responsive header with navigation links
- Main sections: Hero, Features, Team
- Footer with company info

## How to use

Open `index.html` in your browser to view the site.

## Author

Homework for GoIT HTML/CSS course.
